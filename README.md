# Gemini 复制按钮

点击复制整个对话内容

### 安装

Tampermonkey，[点这个](https://github.com/zetaloop/gemini-copy-button/raw/refs/heads/main/gemini-copy-button.user.js)。

### 用法

点击右上角的复制按钮，将会把整个对话的内容按照指定格式复制到剪贴板。

默认格式为：
```markdown
「用户」
用户说的话

========

「助手」
助手说的话

========
...
```
